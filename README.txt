
EDGEHAX DOMAIN-AWARE VOICE CHATBOT

How to Run:

1. Install Ollama from https://ollama.com
2. Run: ollama pull mistral
3. Open this folder in Visual Studio Code
4. Run: pip install -r requirements.txt
5. Run: python app.py

Features:
- Local 7B LLM (Mistral via Ollama)
- Whisper STT
- Text-to-Speech
- RAG using FAISS
- Domain-based answers
